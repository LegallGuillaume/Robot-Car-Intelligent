## Nombres

Valeur signé:

int8_t = valeur de -128 à 127

int16_t = valeur de -32 768 à 32 767

Valeur non signé:

uint8_t = valeur de 0 à 255

uint16_t = valeur de 0 à 65 535

uint32_t = valeur de 0 à 4 294 967 295
